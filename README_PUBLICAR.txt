# Fantamon Boss Builder v2.3 Web

Esta versão foi preparada para publicação como site estático.

## Arquivos para publicar
Envie todo o conteúdo desta pasta para o repositório, mantendo:
- index.html
- script.js
- style.css
- pasta assets
- .nojekyll

## GitHub Pages
1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Abra Settings > Pages.
4. Em Build and deployment, selecione Deploy from a branch.
5. Selecione a branch main e a pasta / (root).
6. Salve e aguarde a publicação.

## Dados dos usuários
Bosses, estratégias e skills adicionadas ficam salvos localmente no navegador de cada pessoa.
Para transferir dados entre computadores ou navegadores, use Exportar Backup e Importar Backup.

## Atualizações
Para atualizar o site depois, substitua os arquivos publicados pelos arquivos da nova versão.
