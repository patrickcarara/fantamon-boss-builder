FANTAMON BOSS BUILDER V5 — RECORTE DO BOSS

Substitua no GitHub:
- index.html
- style.css
- script.js
- github-skills.js

O que esta versão mantém:
- Skills com imagens hospedadas no GitHub
- Botão para remover todas as skills da classe selecionada
- Renomear classe atualizando ID e caminho da pasta
- Recorte da imagem do Boss

Como funciona o recorte:
1. Clique para escolher a imagem do Boss.
2. Abre automaticamente a tela "Recortar imagem do Boss".
3. Arraste a imagem para posicionar o Boss.
4. Use o controle de Zoom.
5. Clique em "Usar recorte".
6. O resultado é salvo em 400 x 560 pixels, comprimido em JPEG para ocupar menos espaço.

Depois de atualizar os arquivos no GitHub Pages, use Ctrl + F5 no navegador.
