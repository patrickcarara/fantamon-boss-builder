FANTAMON BOSS BUILDER V6 — BOSSES PÚBLICOS

COMO FUNCIONA
-------------
Os Bosses que todos os visitantes podem ver ficam em:
data/bosses.json

Quando alguém abre o site, ele baixa automaticamente esse arquivo do GitHub Pages.

COMO PUBLICAR OS BOSSES QUE VOCÊ CRIOU
--------------------------------------
1. Abra o site no computador onde seus Bosses estão cadastrados.
2. Clique em "Gerar Bosses Públicos".
3. O navegador vai gerar um arquivo chamado bosses.json.
4. No seu repositório GitHub, crie a pasta "data" se ela ainda não existir.
5. Envie/substitua o arquivo em:
   data/bosses.json
6. Faça o Commit changes.
7. Aguarde o GitHub Pages atualizar.
8. Abra o site no celular ou em outro dispositivo.

Todos os Bosses publicados no arquivo aparecerão para qualquer visitante.

IMPORTANTE
----------
O GitHub Pages não permite que um site público grave automaticamente no
repositório sem autenticação. Colocar um token do GitHub dentro do JavaScript
seria inseguro, pois visitantes poderiam copiá-lo e modificar seu repositório.

Por isso esta versão usa o método seguro:
- você cria e edita os Bosses;
- gera bosses.json com um clique;
- substitui esse arquivo no GitHub;
- todos passam a ver a atualização.
